# CAPTCHAS.co Drop-in Widget for WordPress

A lightweight, production-ready WordPress plugin that embeds the **CAPTCHAS.co drop-in widget** using an iframe or script loader.

The plugin supports:
- ✅ Shortcodes
- ✅ Gutenberg block
- ✅ Classic Widgets
- ✅ Centralised admin configuration
- ✅ No build tools or dependencies

---

## Features

- **Two embed modes**
  - Iframe embed (recommended)
  - Script-based embed (container + remote script)
- **Multiple placement options**
  - Shortcode: `[captchas_widget]`
  - Gutenberg block: *CAPTCHAS.co Drop-in Widget*
  - Appearance → Widgets
- **Admin settings page**
  - Embed URL / Script URL
  - Width & height controls
  - Accessibility title
- **Server-side rendering**
  - Safe, fast, SEO-friendly
- **Clean uninstall**
  - Removes saved options on uninstall

---

## Installation

### Option 1: Install via ZIP (recommended)

1. Download `captchasco-dropin-widget.zip`
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**
3. Upload the ZIP and activate
